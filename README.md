# 📋 Form Persiapan Event

Form operasional persiapan event berbasis HTML — bisa diisi langsung di browser, lalu cetak atau export ke PDF.

## Fitur

- ✏️ Semua field bisa diedit langsung di browser
- 📅 Kalender interaktif (navigasi hingga tahun 2050)
- 🕐 Pilih jam langsung dari grid atau ketik manual
- ➕ Tambah / hapus baris tabel
- ☑️ Checkbox centang interaktif
- 🖨️ Cetak / Export PDF langsung dari browser
- ↺ Reset form dengan satu klik

## Cara Pakai

### Lokal
1. Download / clone repo ini
2. Buka `index.html` di browser (Chrome / Edge direkomendasikan)
3. Isi form → klik **Cetak / Export PDF**

### Via GitHub Pages
Akses langsung di:
```
https://<username>.github.io/<repo-name>/
```

## Export ke PDF

1. Klik tombol **🖨 Cetak / Export PDF**
2. Printer → pilih **Save as PDF**
3. Pastikan ✅ **Background graphics** aktif
4. Paper size: **A4**, Margins: **Minimum**
